-----------------------------------------------------------------------
Program accepts a directory with any number of nested folders containing data.

All data must be provided as CSV with fixed column order.
 
Each row represents a gamma-decay photopeak.

Program reads values strictly by position, order and number of columns must not be changed.

RedChi
XMax
Energy
EnergyError
Area
AreaError
Intensity
IntensityError

All fields must contain numeric values.
